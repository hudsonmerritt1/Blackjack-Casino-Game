# Blackjack-Casino-Game
Made this for school but expanded it much past what I turned in just for fun :)
•	Command-line Blackjack game with multiplayer, betting systems, and bankroll tracking
•	Full game logic including deck creation, card dealing, hit/stand/surrender decisions, splitting hands, double down, and insurance mechanics
•	Modular Python functions to manage gameplay stages such as player input, dealer behavior, payouts, and game resets
